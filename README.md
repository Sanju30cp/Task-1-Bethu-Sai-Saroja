Data Analytics Project 1-Task 1: Data Cleaning & Preparation
Project Overview
This project was completed as part of the DecodeLabs Data Analytics Internship Program (Batch 20 May - 20 June 2026).
The objective of this project is to clean and prepare a raw e-commerce dataset by identifying and handling missing values, removing duplicate records, correcting data formats, and improving overall data quality.

Dataset Description
The dataset contains e-commerce order information, including:
- Order ID
- Order Date
- Customer ID
- Product Details
- Quantity
- Unit Price
- Total Price
- Payment Method
- Shipping Address
- Order Status
- Tracking Number
- Coupon Code
- Referral Source
The dataset simulates real-world business data with common quality issues such as missing values, duplicates, and inconsistent formatting.


Project Objectives
- Identify and handle missing values
- Remove duplicate records
- Standardize text formatting
- Correct date formats
- Validate numerical data
- Improve dataset consistency and integrity


Data Cleaning Tasks Performed
1. Missing Value Treatment
- Identified null and blank cells.
- Filled missing Coupon Code values with "No Coupon".
- Handled missing values without deleting valid records.

2. Duplicate Removal
- Checked for duplicate Order IDs.
- Removed duplicate records using Excel's Remove Duplicates feature.

3. Date Format Standardization
- Converted all date values into a consistent format.
- Verified that no incorrectly formatted dates remained.

4. Data Validation
- Verified unique Order IDs.
- Checked for remaining blank values.
- Ensured consistency across all columns.

Tools Used
- Microsoft Excel
- Data Cleaning Techniques
- Data Validation Methods

 Key Learnings
